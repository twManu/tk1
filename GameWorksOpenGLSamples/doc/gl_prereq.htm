NVIDIA GameWorks OpenGL Samples Prerequisites

Windows (Target) Development:

  • Windows Vista, 7, or 8 PC
  • NVIDIA GeForce Fermi-based (GTX 4xx) or better GPU
  • Visual Studio 2010 or 2012, non-Express editions
  • Microsoft DirectX End-User Runtime Web Installer. This is required by all of the OpenGL samples on Windows in order to add XINPUT-based game controller support.

Linux x86 (Target) Development:

  • Ubuntu 12.04LTS (later versions, or other Distros, may work but are not officially supported)
  • NVIDIA GeForce Fermi-based (GTX 4xx) or better GPU
  • Your system must have a display driver with full OpenGL support for your graphics card. For GeForce cards, Download and install a recent NVIDIA Linux Display Driver to ensure proper OpenGL support. (The latest NVIDIA Linux x64 driver at time of this writing was the Geforce 331.79 driver, and there are matching driver installation docs should you need them).

Linux ARM/Tegra (Target) Development:

  • Ubuntu 12.04LTS (later versions, or other Distros, may work but are not officially supported)
  • A Tegra K1-based device such as an NVIDIA Jetson TK1, flashed with Linux for Tegra

Android (Target) Development:

  • Windows Vista/7/8 (preferred) or Linux Ubuntu 12.04LTS host PC
  • Tegra 4-based device (advanced samples require a Tegra K1 device)
  • Build environment:
    • Windows, Visual Studio:
      • Visual Studio 2010 or 2012, non-Express editions
      • Full Tegra Android Development Pack for Windows tools install
        (Note that NVIDIA Nsight Tegra, Visual Studio Edition should be included in full install if Visual Studio detected)
    • Linux, command-line:
      • Full Tegra Android Development Pack for Linux tools install
